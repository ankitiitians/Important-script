# Important Scripts

## Overview

This repository contains a collection of essential scripts designed to automate routine tasks, improve operational efficiency, and support system reliability. These scripts are primarily focused on real-world use cases in DevOps, system administration, and infrastructure management.

## Features

* Automation of repetitive system tasks
* Easy-to-use and modular scripts
* Suitable for Linux-based environments
* Helpful for DevOps / SRE workflows

Ensure the following are installed on your system:

* Linux OS (Ubuntu/CentOS preferred)

## Usage

1. Clone the repository:

```
git clone https://github.com/ankitiitians/Important-script.git
cd Important-script
```

2. Give execution permission:

```
chmod +x script_name.sh
```

3. Run the script:

```
./script_name.sh
```

## Example

```
./disk_usage.sh
```
